This is a file uploader written in Python and JavaScript with many nice and comfortable features.

Menu buttons can be triggered with numbers and letters on the keyboard.
For example 1 will trigger the first button.
Ctrl + M
Show the main menu.
Ctrl + Up
Go to the fresh post.
Ctrl + Right
Go to the next post.
Ctrl + Shift + Right
Go to a random post.
Ctrl + Shift + Up
Edit a post.
Ctrl + Down
React to a post.
Ctrl + Dot
Show video commands.
Shift + Wheel on media increases or decreases its size.
Wheel on 'Max' increases or decreases media size as well.
Some buttons open operations in a new tab when middle clicked.
Wheel on an open modal image starts expanded mode.
Image expand and scrolling also work with arrow keys.
The volume button listens to wheel, middle click, and right click.
Long titles can be clicked and expanded to show the full text.
Middle click a reaction icon to use it without closing the menu.
Some popup messages also have middle click actions.
