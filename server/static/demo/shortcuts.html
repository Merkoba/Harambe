Menu buttons can be triggered with numbers on the keyboard.
For example 1 will trigger the first button.
Ctrl + M
Show the main menu.
Ctrl + Up
Go to the fresh post.
Ctrl + Left
Go to the previous post.
Ctrl + Right
Go to the next post.
Ctrl + Shift + Right
Go to a random post.
Ctrl + Shift + Up
Edit a post.
Ctrl + Down
React to a post.
